説明

文字列の連結、挿入、削除や置換を行える文字列バッファ。

String が連結、挿入、削除、置換処理を行うたびに新たにメモリ割り当てを行うのに対し、StringBuffer はバッファサイズが許す限りメモリ割り当てを行わない。 バッファに外部の固定長配列を指定することもでき、この場合は固定長を超えて文字を書き込めない。 String::refer や String::literal と組み合わせることでメモリ割り当てを完全に排除できる。

・サンプルコード


	wchar_t buffer[256];
	StringBuffer s(buffer);
	s += L"abc";
	s += L"def";
	assert(String::equals(buffer, L"abcdef"));
