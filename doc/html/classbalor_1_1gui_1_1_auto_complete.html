説明

エディットコントロールに特定の文字を入力すると文字列の続きの候補を列挙したり補間できるようにする。

・サンプルコード


	Frame frame(L"AutoComplete Sample");
	Edit edit(frame, 20, 10, 0, 0, 10);
	const wchar_t* items[] = {
		L"012345",
		L"012abc",
		L"012あいう",
		L"abc",
		L"abcdef",
		L"あいう",
	};
	AutoComplete complete(edit, items);
	frame.runMessageLoop();
