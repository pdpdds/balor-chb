const String & filter () const
 ダイアログで選択できるファイルの拡張子名と拡張子の組のリスト。拡張子名と拡張子は'
'で区切り、拡張子はセミコロンで区切って複数指定できる。さらに'
'区切りで複数の組を指定する。 "\n\n"で組の終端になる。例えば下記のように指定する。
"すべてのファイル\n*.*\nテキストファイル\n*.txt\nJPEGファイル\n*.jpg;*.jpeg\n\n"
区切り文字は'\0'でも良いが文字列リテラルに'\0'が含まれると String のコンストラクタはそこで文字列が終了していると判断してしまうので String::literal() 関数を使用するかコンストラクタで文字列の長さを指定する事。 初期値は空文字列。
