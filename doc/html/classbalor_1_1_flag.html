説明

template<typename T>
class balor::Flag< T >

ビット演算を bool 値を使ってわかりやすくする。

WIN32 API に多数存在するフラグ定数の操作に使用する。 std::bitset はビット位置をマスクではなく整数位置で指定せねばならず、元の型との相互変換も簡単ではない。
