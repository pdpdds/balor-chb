説明

平面上の位置と大きさを持つ領域を表す構造体。

WIN32 の RECT 構造体と相互変換できる。 WIN32 の Rectangle 関数と名前がかぶるので windows.h と一緒に使う場合は別名の Rect を使うと良い。
