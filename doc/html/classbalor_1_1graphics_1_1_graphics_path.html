説明

描画した一連の線の集合を表す。

GraphicsPath は Graphics クラスの関数と図形の描画結果が微妙に異なるので１ドットの違いも許せない場合には使用すべきではない。 Graphics::drawPath した場合は Graphics::draw〜系関数と結果がだいたい一致するが、Graphics::fillPath した場合は 図形の右下ピクセルを除いた形になるか、図形によっては輪郭部分を除いた形になる。 また GraphicsPath から Region を作成した場合は主に fillPath した場合の領域が作成される。
